## Setup Instructions

1. Clone or download this repository
2. Open `index.html` in a web browser
3. Update the WhatsApp number in `script.js` (line 37)
   - Replace `'1234567890'` with your actual WhatsApp number
   - Format: country code + number (e.g., for UK: 447123456789, for US: 15551234567)

## Publishing to GitHub Pages

1. Create a new repository on GitHub
2. Push all files to the repository
3. Go to Settings > Pages
4. Select the main branch as the source
5. Your site will be live at `https://yourusername.github.io/repository-name/`

## Customization

- Update images: Replace the Unsplash image URLs in `index.html` with your own images
- Change colors: Modify CSS variables in `styles.css` (root section)
- Update content: Edit text in `index.html`
